You can do many powerful things thanks to Arvest’s open RESTful API. A great place to start is to take a look at one of these tutorials.

Arvest API

Arvest's open RESTful API allows you to integrate Arvest into complex machine learning workflows. Consult the full swagger documentation here, and the developer manual here.

We also provide a python package which allows you to easily integrate the API in your python scripts.

ML Notebooks

To help you in your machine learning journey, we are also in the process of building a collection of Machine Learning Notebooks which will take you through a number of typical digital humanities computational workflows. Going through these notebooks, you will learn how to implement a large number of computational techniques, as well as how to integrate Arvest and IIIF into your workflows.
